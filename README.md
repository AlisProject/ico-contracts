# Usage

1. `git clone https://github.com/AlisProject/contracts.git`
1. `cd contracts`
1. `yarn`
1. `yarn truffle install`

# test
- `yarn test`

# debug
- `yarn debug`
    - You can use `debugger;` function of node.
